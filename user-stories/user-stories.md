## User Registration
As a user, I want to be able to register an account so that I can access the platform and use its features.

## User Login
As a user, I want to be able to log in to my account so that I can securely manage my bookings and profile.

## Search Properties
As a guest, I want to search for available properties so that I can find accommodation that meets my needs.

## View Property Details
As a guest, I want to view property details so that I can make an informed decision before booking.

## Book Property
As a guest, I want to book a property so that I can reserve accommodation for my travel dates.

## Make Payment
As a guest, I want to make payments securely so that I can confirm my booking.

## Manage Properties
As a host, I want to add and manage my properties so that guests can book them.

## View Bookings
As a host, I want to view bookings for my properties so that I can manage availability.

## Manage Users
As an admin, I want to manage users so that the platform remains safe and reliable.

## Resolve Disputes
As an admin, I want to resolve disputes so that conflicts between users are properly handled.
